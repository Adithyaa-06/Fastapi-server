# FastAPI JSON Server

A simple FastAPI backend built as part of my AI Engineering internship.

## Features

- GET `/`
- GET `/about`
- Interactive API documentation available at `/docs`

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
uvicorn main:app --reload
```

Open:

- http://127.0.0.1:8000
- http://127.0.0.1:8000/docs
